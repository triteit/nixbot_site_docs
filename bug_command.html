Решил как-то раз я сделать команду для сообщения о баге в личные сообщения. Потому что без этой команды очень трудно передать баг разработчику. В этом гайде я решил показать как это сделать, чтобы Вы не тратили много времени на сложный код.

Итак, что нам для этого надо? Для команды нам надо создать файлы                     и                            в папке нашего бота, далее создаём 3 функции, первая которая будет считывать количество жалоб, ​делаем это для того, чтобы после каждого перезапуска, количество жалоб не приводилось заново к нулю, вторая функция создана для добавления 1 единицы жалобы, как только пользователь отправил данную команду, а третья функция будет сохранять сообщения, номер жалобы и дискорд отправителя в файле                    .

def get_current_report_number ():
    try :
        with open ( 'report_number.txt' , 'r' ) as file :
            report_number = int ( file . read ( ) )
    except ( ValueError, FileNotFoundError ) :
        report_number = 1
    return report_number 

Вот подробный код, как реализовать первую функцию:

def increment_report_number ():            
     report_number =  increment_report_number ()
    report_number += 1
    with open ('report_number.txt' , 'w') as file:
        file. write ( str ( report_number ))

А вот код для второй функции, которая будет добавлять единицу, как только пользователь отправил данную команду:

def save_report (report) :            
    with open ('report_number.txt' , 'w') as file:
        file. write (report +   ' \​n' ))

И ещё один код, но уже 3 функции, которая будет записывать сохранять сообщение, номер жалобы и дискорд отправителя в файле                    .
